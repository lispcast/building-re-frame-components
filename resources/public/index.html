Building Re-frame Components

Welcome to Building Re-frame Components. You can find this course online. You need to install Leiningen. You should then run Figwheel in this project's directory like this: 


lein figwheel

That will begin watching changes to the code files in the src/ directory. As you save changes to the files, you can watch them change in your browser. Click on one of the lessons below (Teacher is my version of the code, Student is yours). Then open the cljs file in the editor of your choice, make some changes, and watch them change in the browser.

If you have any questions or you find a bug, I am more than welcome to help you. Please contact me directly at eric@lispcast.com.

Rock on!
Eric
